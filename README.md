# 文本处理工具
tools目录中是网上收集的文本去重工具，可自由选择使用那个

去除指定域名的正则:
(.*?)\.xxx\.com([^.A-Za-z0-9])

此正则将查找所有以 xxx.com的域名结尾的行， 将此替换为空即可去除指定的域名的屏蔽
